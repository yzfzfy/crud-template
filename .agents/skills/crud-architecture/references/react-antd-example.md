# React + Ant Design CRUD Implementation Example

**触发条件**：仅当用户明确要求 “使用 React”、“React + Antd”、“React 版本” 等时，才使用本示例。

**入口文件说明**：主列表页面使用 `index.tsx`（符合很多 React 项目习惯）。

## 目录结构（必须严格遵守）

```bash
user-management/

├── form-modal/
│   ├── index.tsx
│   ├── modal-content.tsx
│   └── types.ts

```
